# BagAura – Handbag Product Page

BagAura is a static e-commerce product page designed to showcase a women’s premium handbag. The project focuses on clean UI design, accessibility, and real-world e-commerce layout practices.

## Features
- Professional product page layout
- Main product image with clickable thumbnail previews
- Pricing displayed in Indian Rupees (INR)
- Discount and product details section
- Quantity selector with validation
- Add to Cart button
- Accessibility-compliant design

## Technologies Used
- HTML5 for page structure
- CSS3 for styling and layout
- JavaScript for interactive features

## Project Structure
handbag-product-page/
│
├── index.html
├── styles.css
└── script.js

## How to Run
1. Ensure all files are in the same directory
2. Open index.html in a web browser
3. Use Live Server in VS Code for better development experience

## Functionality Overview
- Clicking thumbnail images updates the main product image
- Quantity buttons increase or decrease the product quantity
- JavaScript handles all dynamic interactions
- This is a frontend-only project

## Accessibility and Best Practices
- All images include appropriate alt text
- Form inputs and icon buttons use ARIA labels
- Proper input types and validation are applied
- Code is clean, readable, and maintainable
